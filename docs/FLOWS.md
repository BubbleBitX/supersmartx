# Core Flows

## Create first post
1. Open `Create`
2. Enter milestone details
3. Review preview and caption
4. Export or save

## Reuse saved post
1. Open `Timeline` or `Dashboard`
2. Select a saved item
3. Open `Create` with prefilled content
4. Edit and export

## Complete profile
1. Open `Profile`
2. Save name, role, company, and photo
3. Return to `Create`
4. See defaults auto-filled

## Upgrade intent
1. Open `Pricing`
2. Review plan differences
3. Use contact CTA while payments are disabled
4. Convert later after billing rebuild

## Required states
- Empty
- Loading
- Success
- Error
- Permission denied
- Rate limited
- Offline export is unavailable because persistence and entitlements are server-backed
