# SuperSmartX PRD

## Product
SuperSmartX is a web app for creating polished career milestone graphics and platform-specific captions, with LinkedIn as the primary launch wedge.

## Launch position
- Core wedge: career and job milestones to LinkedIn-ready posts
- Core promise: set up your profile once, reuse it for every future post
- Free plan: create and export basic watermarked posts
- Paid expansion: unlock premium exports and premium template access after a secure billing rebuild

## Primary users
- Job seekers sharing offers, internships, and shortlist updates
- Professionals sharing promotions, certifications, and role changes
- Founders and operators sharing public business milestones

## Launch goals
- Let a new user create one polished post in under 5 minutes
- Keep copy simple, low-noise, and task-focused
- Make saved profile and reuse flow obvious

## Non-goals for launch
- Live analytics integrations
- Team accounts
- Marketplace features
- AI image generation
- Production billing until secure checkout and webhook validation exist
